# PySimplex2D
A visualisation of the primal simplex algorithm in $\mathbb R^2$

## Problem Format
- Objective: $\max c \cdot x$
- Constraints: $Ax \leq B$

## Usage
- Enter your problem data in `simp_user.py` and run it
- Press spacebar to step forward
